Importe um arquivo .xlsx com a lista de atividades. As atividades atuais desta obra serão substituídas — o progresso (status) é preservado para os IDs que continuarem existindo.
O que o sistema faz sozinho na importação:
✓ Identifica Projeto ou Obra automaticamente pelo nome/departamento (você só precisa informar se quiser sobrescrever)
✓ Calcula o caminho crítico com base nas datas (cadeia sem folga)
✓ Gera IDs faltantes, completa os dias e valida sobreposições
✓ Mantém o status das atividades cujos IDs continuarem existindo

Vamos criar sua construtora
Cada construtora tem seu próprio painel — obras, equipe e marca compartilhados entre todos os usuários da empresa. Você é o primeiro, então será o dono (owner) e poderá adicionar outros usuários depois.
Cadastrar usuário da equipe
Você está criando um login direto para um funcionário da sua construtora. Ele(a) vai receber acesso ao painel com a senha que você definir aqui — passe pra ele(a) por um canal seguro. O usuário pode trocar a senha depois pelo Firebase (em uma versão futura, vamos abrir essa tela aqui).
Mínimo 6 caracteres. O Firebase aceita qualquer combinação; recomende ao usuário trocar no primeiro acesso.
